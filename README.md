# task-5-pbi-btpns-AGUS-HERYANTO

Cara Menggunakan :

- create database MYSQL
- Siapkan environment variable dibawah ini:
- MYSQL_DBNAME
- MYSQL_HOST
- MYSQL_USER
- MYSQL_PASSWORD
- MYSQL_PORT
- SERVER_PORT
- SERVER_HOST
- Jalankan : migrate -database 'mysql://USER:PASSWORD@tcp(HOST:PORT)/DBNAME' -path database/migrations up

Setelah itu

- Jalankan : go run .
